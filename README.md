# Projeto de Laboratório de Git

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de versionamento utilizando Git e GitHub.

## Aprendizados

- Uso de branches
- Criação de commits
- Utilização do GitHub Flow
- Resolução de conflitos

## Conflito

Foi simulado um conflito no arquivo README.md ao modificar a mesma linha em branches diferentes. O conflito foi resolvido manualmente.
